# 템플릿 기반 학원 학부모 문자 자동완성 웹사이트

https://smsgenerator.netlify.app/

국어학원에서 조교 업무를 하며 조교의 본질인 '학생에게 양질의 피드백을 제공하는 것'에 더 집중할 수 없음에 불편함을 느끼고, 반복되는 업무를 자동화할 수 있는 웹페이지를 설계 및 제작함.

처음에는 엑셀을 활용해 비슷한 기능을 구현해 사용했지만, 매주 페이지 새로 복제 - 기존 내용 삭제하는 번거로움 + 수식 수정의 위험이 있다는 점이 항상 아쉬웠음. 

이에 기존 엑셀 수식을 기반으로, figma make와 vscode를 활용해 웹사이트를 구현함. 개발수업에서 배운 무료 배포사이트 netlify를 통해 배포하고, 함께 근무한 3명의 조교들과 함께 사용함.
